# 주식 테마 이해도 퀴즈 앱

## 1. 프로젝트 소개

이 프로젝트는 Python과 Streamlit을 활용하여 만든 주식 테마 퀴즈 웹 애플리케이션입니다.  
사용자는 로그인 후 주식 시장에서 자주 등장하는 기업, 테마, 기술 키워드 관련 퀴즈를 풀 수 있습니다.

현재 퀴즈 주제는 기업의 기술 키워드와 관련 테마를 이해하는 방식으로 구성되어 있습니다.  
예를 들어 케이씨에스의 양자암호원칩 관련 기능과 협력 통신사에 관한 문항이 포함되어 있습니다.

> 본 앱은 학습용 퀴즈 앱이며, 특정 종목의 매수·매도 추천을 목적으로 하지 않습니다.

## 2. 제출자 정보

- 학번: 학번을_입력하세요
- 이름: 이름을_입력하세요

## 3. 주요 기능

### 첫 화면 정보 표시
앱 첫 화면에 제출자의 학번과 이름을 표시합니다.

### 로그인 기능
미리 정의된 아이디와 비밀번호를 입력하면 로그인할 수 있습니다.

테스트 계정:

| 아이디 | 비밀번호 |
|---|---|
| student | 1234 |
| stock | 2024 |
| guest | 0000 |

### 캐싱 기능
Streamlit의 `@st.cache_data`를 사용했습니다.

캐싱 적용 위치:

1. `load_quiz_data()`
   - CSV 파일에서 퀴즈 데이터를 불러오는 함수입니다.
   - 퀴즈 문항은 앱 실행 중 반복 사용되므로 캐싱을 적용해 불필요한 파일 읽기를 줄였습니다.

2. `calculate_result()`
   - 정답 개수에 따른 결과 유형을 계산하는 함수입니다.
   - 같은 점수에 대해 같은 결과가 반복되므로 캐싱을 적용했습니다.

### 퀴즈 기능
사용자는 주식 테마 관련 객관식 퀴즈를 풀고, 최종 정답 수에 따라 결과 유형을 확인합니다.

결과 유형 예시:

- 테마 분석 고수형
- 테마 관심 성장형
- 기초 탐색형

## 4. 실행 방법

### 1) 저장소 클론

```bash
git clone 저장소주소
cd 저장소폴더
```

### 2) 가상환경 생성 및 활성화

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

macOS/Linux:

```bash
source venv/bin/activate
```

### 3) 패키지 설치

```bash
pip install -r requirements.txt
```

### 4) 앱 실행

```bash
streamlit run app.py
```

## 6. 데모 영상 설명 예시

1. 앱을 실행하면 첫 화면에 학번과 이름이 표시됩니다.
2. 로그인 영역에서 테스트 계정으로 로그인합니다.
3. 로그인 성공 시 주식 테마 퀴즈 화면이 나타납니다.
4. 퀴즈 데이터는 CSV 파일에서 불러오며, `@st.cache_data`를 적용해 반복 로딩 속도를 개선했습니다.
5. 사용자가 문항에 답하면 정답 여부가 계산됩니다.
6. 최종 정답 수에 따라 주식 테마 이해도 유형과 추천 학습 방향이 출력됩니다.
